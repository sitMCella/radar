# Radar Hub

## Introduction

This project consists of a Spring Boot application for configuring the Radar Hub.

## Development

### Requirements

- OpenJDK 21

### Build Application

```sh
./gradlew clean build
```

### Run Application

```sh
java -jar build/libs/radarhub-0.0.1-SNAPSHOT.jar
```

### Format Code

```sh
./gradlew :spotlessApply
```
